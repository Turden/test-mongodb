# Instruction

## Requirements

```
Nodejs >= 8.1.2 https://nodejs.org/en/
```
MongoDB >= https://www.mongodb.com/download-center?jmp=nav#community
```


## Clone

```
$ git clone git@github.com:Turden/Test-node-js.git
```

## Setup external dependencies
```
$ mongod

```
$ npm i
```

## Usage

```
$ npm run main
```
